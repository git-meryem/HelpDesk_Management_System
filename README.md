
HelpDesk Management System
A web-based HelpDesk application built with **ASP.NET MVC** for managing support tickets efficiently between users, agents, and administrators.

  Features :

- Authentication (Login / Register / Forgot password)
-  Role-based system (Admin / Agent / User)
-  Create, update, and track tickets
-  Admin & Agent Dashboard
-  Ticket status (Open / In Progress / Closed)
-  Priority levels (Low / Medium / High)
-  Comments system (discussion per ticket)
-  File attachments support
-  Ticket assignment to agents
-  Statistics and charts (Dashboard analytics)

  Tech Stack :

- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- Bootstrap 
- JavaScript
- Chart.js
